## 仕様

#### `merge_sort(a: Iterable[T], key: Callable[[T, T], bool]=lambda s, t: s < t) -> List[T]:`

- 列 `a` をマージソートします。
- 比較関数 `key` にしたがって比較演算をします。
- 時間計算量 `Θ(nlogn)` です。
