class algorithm.doubling.Doubling(n: int, lim: int, move_to: Callable[[T], T])[ソース]

ベースクラス: Generic[T]

ダブリングテーブルを構築します。 \(O(n\log{lim})\) です。

パラメータ:

  • Generic (T) -- 状態の型です。

  • n (int) -- テーブルサイズです。

  • lim (int) -- クエリの最大数です。

  • move_to (Callable[[T], T]) -- 遷移関数です。 u から v へ遷移します。

kth(start: T, k: int) → T[ソース]

start から k 個進んだ状態を返します。 \(O(\log{k})\) です。

パラメータ:

  • start (T) -- スタートの状態です。

  • k (int) -- 遷移関数を適用する回数です。
