仕様

class algorithm.random.random.Random[ソース]

ベースクラス: object

乱数系のライブラリです。 標準ライブラリよりも高速なつもりでいます。

randint(begin: int, end: int) → int[ソース]

begin 以上 end 以下 のランダムな整数を返します。 \(O(1)\) です。

制約:

\(begin \leq end\)

random() → float[ソース]

0以上1以下の一様ランダムな値を1つ生成して返すはずです。 \(O(1)\) です。

randrange(begin: int, end: int) → int[ソース]

begin 以上 end 未満 のランダムな整数を返します。 \(O(1)\) です。

制約:

\(begin < end\)

shuffle(a: List[Any]) → None[ソース]

a をインプレースにシャッフルします。 \(O(n)\) です。

パラメータ:

a (List[Any]) -- a をシャッフルします。
