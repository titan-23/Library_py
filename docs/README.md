# [Library_py](https://github.com/titanium-22/Library_py)

競技プログラミング用のライブラリです。pypyで動きます。色々と更新しています。  
各REAMDMEはサボっています。また、ありえないバグを仕込んでたりします。ご注意ください。  
誰かがバグに引っかかってWAを出してくれたら、筆者冥利に尽きます。
